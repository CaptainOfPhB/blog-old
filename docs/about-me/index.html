关于我

Updated on 2022.02.16

我是船长，常用的英文 ID 是 CaptainOfPhB。

2016 年我于西安理工大学毕业之后在老家国企移动上班，奈何对腐朽的办公室政治忍无可忍，于是在一年半后毅然辞职，只身一人前往上海，我曾写了一篇文章用来记录这段往事。目前我从事于前端开发工作。

我养了两只很可爱的猫，一只狸花叫球球，另一只美短叫小胖。如果你想和它俩一起玩耍，可以添加页面右上角我的微信。

我没什么伟大的理想，但是活着也还算有趣。

如果你能为我提供一份好的工作，这是我的简历，你可以在网页上右键点击打印。

祝你身体健康，有个好心情。
